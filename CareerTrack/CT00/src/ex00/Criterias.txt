Критерии:
- Заработная плата: выучить необходимый стек на позицию junior data scientist, выпуститься из университета, сделать пару проектов на kaggle, пройти успешно интервью, принять оффер на работу;
- Удалённая работа: посмотреть доступные вакансии, выбрать те вакансии, которые предоставляют удалённую работу, устроиться по одной из этих вакансий;
- Профессиональный рост: изучать базовый стек ds-ера, читать углублённую литературу по ML, изучать новинки ML, следить за новостями мира Data Science, закончить обучение в школе 21, закончить дипломную работу;
- Престиж компании: выписать список престижных компаний, посмотреть доступные вакансии компаний из списка, пройти интервью по данным вакансиям, принять оффер на работу из престижной компании;
- Цели проектов: устроиться на работу, познакомиться со своей командой, узнать цели разрабатываемых проектов, оставить команду или попросить перенаправить в другую команду разработчиков из-за неподходящих целей проектов;
- Акционерность: выписать список акционерных компаний, посмотреть вакансии акционерных компаний из списка, пройти интервью и устроиться на работу по вакансии из списка;
- Вклад в развитие мира: определить интересные компании направленные на развитие мира, узнать цели компаний, отобрать вакансии компании с интересными целями, принять участвие в собеседовании по отобранным вакансиям, принять оффер на работу;
- Интересность: изучить раздел "чем предстоит заниматься" вакансий интересующий компании, выбрать подходящую вакансию, участвовать в собеседовании по интересующей вакансии, 
принять оффер;
- Владение выполненными проектами: изучить конфиденциальность компаний по разрабатываемым продуктам, выбрать подходящие компании по вакансиям, принять участие в собеседовании по отобранной вакансии, принять оффер;
- График работы: рассмотреть вакансии с подходящим пунктом "график работы", выписать список подходящих вакансий, откликаться по вакансиям до первого успеха, пройти собеседование по вакансии, принять оффер.
